# 📊 Zohan Store Performance Dashboard (2022 Data):<br>
      
       An interactive Excel Sales Analytics Dashboard built to practice analyzing a retail store's yearly <br>
       performance across sales, orders, customer demographics, and sales channels.<br>

# 📅 Note: This is a practice/portfolio project using 2022 sample retail data,<br>
  #         created to demonstrate Excel dashboarding and data analysis skills.<br>

# 🎯 Project Objective:<br>

   This project analyzes a retail store's 2022 sales data to uncover actionable<br>
   business insights — customer behavior, top-performing regions, and best sales channels.<br>
   It was built as a hands-on exercise to strengthen Excel dashboarding and data visualization skills.<br>
  
# 🛠️ Tools Used:<br>

   Microsoft Excel.<br>
   Pivot Tables & Pivot Charts.<br>
   Slicers (Interactive Filters).<br>
   Data Visualization (Bar, Line & Pie Charts).<br>

# ❓ Sample Business Questions Answered:<br>

   📈 Sales vs Orders trend comparison.<br>
   📅 Highest sales/orders month.<br>
   👫 Men vs Women purchase comparison.<br>
   📦 Order status distribution.<br>
   🗺️ Top states contributing to sales.<br>
   🧓 Age group vs gender order relationship.<br>
   🛒 Best performing sales channel.<br>
   👕 Highest selling product category.<br>

# 💡 Key Insights:<br>
#    InsightDetail:<br>
      
   👩 Gender Split   Women contribute ~64% of total sales.<br>
   🗺️ Top States     Maharashtra, Karnataka & Uttar Pradesh.<br>
   👨‍👩‍👧 Age Group      Adults (30-49 yrs) drive max orders.<br>
   🛒 Top Channels   Amazon, Flipkart & Myntra (~80% sales).<br>

# 🚀 Final Recommendation:<br>

Target women customers aged 30-49 yrs in Maharashtra, Karnataka & Uttar Pradesh<br>
through targeted ads/offers/coupons on Amazon, Flipkart & Myntra to maximize sales growth.<br>
