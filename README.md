# 📊 Zohan Store Performance Dashboard (2022 Data):
      
       An interactive Excel Sales Analytics Dashboard built to practice analyzing a retail store's yearly<br>
       performance across sales, orders, customer demographics, and sales channels.

# 📅 Note: This is a practice/portfolio project using 2022 sample retail data,
  #         created to demonstrate Excel dashboarding and data analysis skills.

# 🎯 Project Objective:

   This project analyzes a retail store's 2022 sales data to uncover actionable<br>
   business insights — customer behavior, top-performing regions, and best sales channels.<br>
   It was built as a hands-on exercise to strengthen Excel dashboarding and data visualization skills.
  
# 🛠️ Tools Used:

   Microsoft Excel.
   Pivot Tables & Pivot Charts.
   Slicers (Interactive Filters).
   Data Visualization (Bar, Line & Pie Charts).

# ❓ Sample Business Questions Answered

   📈 Sales vs Orders trend comparison.
   📅 Highest sales/orders month.
   👫 Men vs Women purchase comparison.
   📦 Order status distribution.
   🗺️ Top states contributing to sales.
   🧓 Age group vs gender order relationship.
   🛒 Best performing sales channel.
   👕 Highest selling product category.

   💡 Key Insights
      InsightDetail
      
👩 Gender Split   Women contribute ~64% of total sales
🗺️ Top States     Maharashtra, Karnataka & Uttar Pradesh
👨‍👩‍👧 Age Group      Adults (30-49 yrs) drive max orders
🛒 Top Channels   Amazon, Flipkart & Myntra (~80% sales)
